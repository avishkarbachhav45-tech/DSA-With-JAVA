# DSA With Java

This repository contains programs and practice questions related to **Java fundamentals** and **pattern printing**.  
All files are neatly organized into topic-wise folders for easy understanding.

---


---

### **01-Java-Basics/**
Contains beginner-friendly Java programs and concepts:
- Variables, Datatypes  
- Operators  
- Conditional Statements  
- Loops  
- Functions  
- Boilerplate code examples

Basic pattern programs such as:

- Star Patterns  
- Character Patterns  
- Nested Loop Concepts

### **03-Patterns-Part-II/**
Advanced pattern printing:
- Butterfly Pattern

- 0-1 Triangle
